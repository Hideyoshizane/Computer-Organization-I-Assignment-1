# Computer Organization I Assignment 1

This assignment objective was the implementation of a given problem in Assembly Risc-V.
The project was divided in 2 Problems. The first one was to reverse a vector using recursion, the vector was reversed using the outside to inside method.
The second Project was to implement a program that verify if a given CPF number, the Brazilian registry identification, or a CNPJ, Brazilian Registry of Legal Entity, is a valid number.
